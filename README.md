# Windrive
Windows Drive (disk) & Devices - Node.JS low level binding

## Requirement

- Windows build tools
- Node-gyp

Take a look at: https://www.npmjs.com/package/windows-build-tools

## Starter guide

To setup the project on your computer

```bash
$ npm install
$ node-gyp configure
$ node-gyp build
```
